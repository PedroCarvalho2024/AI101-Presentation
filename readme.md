# AI 101 Presentation Support Repository

Welcome to the support repository for the **AI 101: From Foundations to Agentic Systems** presentation.

## Overview
This repository contains all supporting materials and artifacts for the AI 101 presentation. The main presentation content is located in the `ai-artifacts` directory.

## Repository Structure
- **ai-artifacts/**
  - `1 - AI 101_ From Foundations to Agentic Systems.md`: The main presentation document.
  - `slide_content.md`: Additional slide content or notes.

## Purpose
This repository is designed to provide easy access to the presentation materials for attendees and anyone interested in learning about the foundations and evolution of AI towards agentic systems.

## How to Use
- Browse the `ai-artifacts` folder to find the main presentation and supporting content.
- Use these materials for reference, study, or as a template for your own AI presentations.
